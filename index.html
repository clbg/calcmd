Why CalcMD?

AI assistants generate tables with numbers every day. CalcMD makes those numbers trustworthy.

🔍

Verifiable

Formulas are embedded in plain text. Anyone can check the math — no spreadsheet required.

🤖

AI-friendly

Column-name formulas like Total=Qty*Price are easy for LLMs to generate and validate.

📝

Human-readable

No A1 cell references. Formulas read like plain English and degrade gracefully in any markdown viewer.

🌿

Git-friendly

Plain text format. Formula changes show up cleanly in diffs. Works everywhere markdown works.

🔒

Secure

Whitelist-only functions, sandboxed evaluation. No scripting, no file access, no surprises.

⚡

Lightweight

Not a spreadsheet. Not a notebook. Just tables with formulas — simple by design.

Syntax at a glance

Two places to put formulas. That's it.

Column header formula — applies to every row

| Item | Qty | Price | Total=Qty*Price |
|------|-----|-------|-----------------|
| Apple | 3 | 1.50 | 4.50 |
| Banana | 5 | 0.80 | 4.00 |

Cell formula — typically used for aggregations

| Total | | | 8.50=sum(Total) |

Supported functions

# Aggregations
sum(col)   avg(col)   min(col)   max(col)   count(col)

# Math
round(n, d)   abs(n)   floor(n)   ceil(n)

# Conditional
if(Score>=90, "A", "B")
